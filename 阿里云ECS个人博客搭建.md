# 阿里云ECS个人博客搭建
前置准备（先把阿里云试用 ECS 配置好）
1、ECS 必做配置
进入阿里云 ECS 实例 → 重置实例密码（root 账号密码，自己记牢）
重置密码后 重启 ECS 实例
复制保存：公网 IP、root 用户名、root 密码
2、放行阿里云安全组（企业最小权限原则）
ECS 控制台 → 网络与安全 → 安全组 → 配置入方向规则
| 协议 | 端口 | 授权对象 | |说明
|-------|-------|-------|-------| 
| TCP | 22 | 本地电脑ip | 只允许自己 SSH 连接，防暴力破解 |
| TCP | 80 | 0.0.0.0/0 | 网站 HTTP 访问 |
| TCP | 443 | 0.0.0.0/0 | 只网站 HTTPS 访问 |
